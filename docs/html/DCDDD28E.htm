Albatross.CodeGen Namespace

Base namespace for the Albatross.CodeGen assembly. The namespace contains interfaces, attributes, types and exceptions defined to create a set of code generation patterns. It contains the implentations for CompositeCodeGeneratorT, O and StaticCodeGenerator. Other notible implmentations are CodeGeneratorFactory, SourceTypeFactory and OptionTypeFactory

Branch
This class represent a branch object for the composite code generation. The object itself is an INode and it can contain other nodes.
ClassGeneratorT
ClassGenerator is an abstract class for C# class code generation. It uses a fixed option type ClassOption but the Source Type is left open. It will take care of namespace, class scope, class name and constructor generation using the information provided by the ClassOption object. It leaves the rest of the body creation to a abstract method RenderBody(StringBuilder, Int32, T, ClassOption).

ICodeGeneratorT, O
This interface is responsible for all code genration implementation. A code generator requires two pieces of data. 1: source; 2: options. The source indicates what the generator is running against such as a database table or a .net type. The option indicates the supported options for the particular generator. It could be the name of the generated class or whether the javascript output should use camel case. CodeGenerator implementation are not expected to be thread safe, since new instances will be created for each execution. Instances of this interface will be created using an Ioc container. Therefore its dependencies should be declared in its constructor. Only a single constructor is supported.
